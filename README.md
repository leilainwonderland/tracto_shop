# tracto_shop


Votre client a besoin de : 
    - D'un site de vente permettant d'écouler son stock de Tractopelles normalisés et identiques. 

Il doit être possible: 
    Back-Office :
        - Créer des marchandises (stock, prix, nom, image)
        - Editer des marchandises (nom, prix)
        - Supprimmer des marchandises
    Front-Office
        - Page d'acceuil qui vous donne les marchandises du momment
        - Page des marchandises qui liste les marchandises, si il en reste en stock et leur prix. 
        - Page pour voir les détails d'une marchandise
        - Système pour commander la marchandise (retirer dans le stock) 
        ajouter une commande avec la date et quoi a été commandé
